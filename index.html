Project on Console

To setup Visual Code first go to Visual Code and download the software and install it


Console screen is a screen that is used to store some extra informartion about the website used by the developer of that website.Steps to open console screen:

1. Right click on your screen.
2. Click on inspect option.
3. Now click on console on the top-right corner of the screen.
4. Now you can see what is written in the console.
